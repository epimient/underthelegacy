# Plan de Mejoras Web – Under The Legacy

## Objetivo Principal

Transformar la web de Under The Legacy de un sitio informativo a una plataforma de construcción de comunidad y validación de mercado.

Actualmente la web presenta correctamente la banda.

El siguiente paso es lograr que cada visitante deje algún dato de contacto o manifieste interés en un producto futuro.

---

# Diagnóstico Actual

Actualmente la web permite:

- Conocer la banda.
- Escuchar la música.
- Ver información general.
- Ver el merchandising conceptual.
- Contactar a la banda.

Pero no permite responder preguntas estratégicas como:

- ¿Quién visita la página?
- ¿Cuántos quieren comprar el EP?
- ¿Cuántos quieren un vinilo?
- ¿Cuántos quieren una camiseta?
- ¿Cuántos quieren recibir noticias?

La consecuencia es que cada visita se pierde.

# Filosofía

La web no debe existir únicamente para mostrar la banda.

La web debe convertirse en el lugar donde Under The Legacy construye una comunidad propia, mide demanda real y prepara el terreno para futuros lanzamientos físicos como el vinilo.
